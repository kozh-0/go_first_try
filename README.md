# Golang try

package - набор го файлов в папке  
module - коллекция packages

`go mod init kek` - инициализация go проекта (создаст go.mod, типа package.json)  
Обычно называют ссылкой на github repo

В каждой папке все файлы начинаются с названия package  
package main - специальная точка входа, прога будет ее запускать первым делом

`go build ./cmd/test/main.go` - создаст бинарный файл, который можно исполнить `./main`  
`go run ./cmd/test/main.go` - создаст и исполнит

Важно выделять память на переменные, чтобы достигать лучшей производительности (например в 3 раза быстрее)
